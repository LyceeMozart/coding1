1989:La naissance de Web

Internet est une infrastructure de réseau munie de moyens de transmission et de moyens logiciels permettant à des machines de communiquer entre elles.

Les trois caractéristiques essentielles Web

Le protocole HTTP
est l'ensemble des règles permettant de transférer des fichiers tels que du texte, des images, du son, de la vidéo et d'autres fichiers multimédias sur le Web
Le langage HTML
est le langage utilisé pour structurer une page web et son conten
Le système URL
est simplement l'adresse d'une ressource donnée, unique sur le Web.
